# MTG deck editor backend
this is the backend for my MTG deck editor

## Made with
- Express.js
- NPM
- MongoDB (NoSQL)

## Link to frontend code
https://github.com/jon2502/mtg-deck-editor

## Setup to dev
1. Clone code
2. Setup .env
3. Run <code>npm install</code>

Localhost is set to 3500

## .env
```env
MONGODB_URI
MONGODB_NAME
```
